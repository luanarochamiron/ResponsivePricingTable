# Responsive Pricing Table

Uma seção de tabela de preços (Pricing Table) moderna, intuitiva e totalmente responsiva desenvolvida com HTML5 e CSS3. O projeto organiza três planos de serviços (Basic, Extended e Premium) em cartões individuais e estruturados, utilizando uma paleta de cores distinta para categorizar e diferenciar cada nível de assinatura.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica dos cartões de planos, listas de recursos e botões de conversão.
* CSS3: Estilização completa, uso de gradientes e gerenciamento de formas geométricas personalizadas.
* Flexbox / CSS Grid: Alinhamento responsivo dos cartões em grade e organização interna linear dos recursos de cada plano.

## Funcionalidades e Técnicas Aplicadas

* Cabeçalhos com Formato em Seta (Ribbons): Áreas superiores dos planos customizadas com um formato angular de seta apontando para baixo, desenhadas através de manipulação de bordas ou pseudo-elementos em CSS puro.
* Identificação Cromática dos Planos: Uso de cores estratégicas para segmentação visual (Verde para Basic, Laranja para Extended e Ciano para Premium).
* Indicadores Visuais de Recursos: Listas de especificações acompanhadas por ícones vetoriais de verificação (✔️) para itens inclusos e marcas de exclusão (❌) para funcionalidades ausentes no plano.
* Precificação Destacada: Exibição centralizada dos valores monetários com fontes robustas e estilização diferenciada para os centavos em formato sobrescrito.
* Chamadas para Ação Simétricas: Botões inferiores personalizados ("Purchase") alinhados de forma idêntica em todos os cards para garantir consistência visual.
